# Solvimm - Estágio em Dados

Desafio de dados para oportunidade de estágio. O desafio consiste em uma simulação de estudo de mercado para formular a estratégia de negócio que o cliente irá adotar.
Diante disso, foram analisados dois datasets (movies.csv e customers_rating.csv) com a biblioteca ***pandas*** do python respondendo as perguntas abaixo.
- 1.1. Quantos filmes estão disponíveis no dataset?
- 1.2. Qual é o nome dos 5 filmes com melhor média de avaliação?
- 1.3. Quais os 5 anos com menos lançamentos de filmes?
- 1.4. Quantos filmes que possuem avaliação maior ou igual a 4.7, considerando apenas os
filmes avaliados na última data de avaliação do dataset?
- 1.5. Dos filmes encontrados na questão anterior, quais são os 10 filmes com as piores notas e
quais as notas?
- 1.6. Quais os id's dos 5 customer que mais avaliaram filmes e quantas avaliações cada um
fez?

### Como executar o código do zero

Nesse mesmo repositório, basta acessar o arquivo "Solvimm.ipynb" e em seguida em "Open in Colab".

Você será redirecionado para o Google Colab da implementação.
Em seguida, apenas pressione "Ctrl" + F9 para rodar todas as células sequencialmente. Todas as explicações das linhas de código estão feitas como comentários em cada célula.

Cada célula está indexada com a pergunta correspondente.
